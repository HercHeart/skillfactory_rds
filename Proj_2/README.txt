→ Oсновные цели и задачи проекта
Провести EDA анализ датасета с оценками по математике для дальнейшего построения модели предсказания отставания учеников по математике

→ Краткую информацию о данных
Данные содержат в себе различную информацию об учениках, составе семьи, успеваемости, времени на дорогу и других параметрах, которые косвенно или прямо могут быть связаны с успеваемостью

→ этапы работы над проектом.
Работа выполнялась в 3 этапа:
- Подготовка, осмотр данных, поиск ошибок, заполнение пропусков.
- Проверка статистичексих гипотез о влиянии различных признаков на оценку
- Основные выводы проекта
